# DeepSeek API简单实践
**此项目是Ai网站对接DeepSeek大模型**
## 第一次完整的做全栈，对过程进行简单记录。

## 1. 文件结构
- `readme.md` ------ 文档
- `app.py` ------ 后端服务文件
- `static/` ------ 静态资源文件夹
  - `static/css` ------ 样式文件
  - `static/img` ------ 网页图片资源
  - `static/js` ------ 前后端交互
  - `static/js/index.html` ------ 布局文件
- `.env` ------ 存放 DeepSeek 的 API Key
